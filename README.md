# Cick Demo

A small demo of the command-line interface framework [Click](https://click.palletsprojects.com/en/stable/), following their documentation, for the TIMESTEP Astro SE Internship, 2025-2026. 

## Installation

This demo uses `uv` for package management. After cloning the repository, run

`$ uv tool install . -e`

from the top-level directory to install the CLI.
